# School Management System (C++ & OOP)

### 🚀 Overview
A terminal-based administrative system built to automate academic workflows. This project was developed to demonstrate **Logical Building** and **Data Persistence** using C++ without external databases.

### 🛠 Technical Features
* **OOP Architecture:** Managed Students, Teachers, and Courses using classes, inheritance, and encapsulation.
* **File-Based Persistence:** Implemented `fstream` for permanent data storage and retrieval.
* **Relational Logic:** Built systems to link Students to specific Sections and Courses.

### 💻 How to Run
1. Clone the repository.
2. Compile `oopproject.cpp` using any C++ compiler (GCC/Visual Studio).
3. Run the executable and follow the on-screen menu.
